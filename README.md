# Eldor Trade Lanes — Logistics Carbon Routing v2

Interactive carbon emission calculator for Eldor's logistics routes between China, Turkey, Italy, and Germany.

## Features
- Sea routes: Suez Canal, Cape of Good Hope
- Rail: Trans-Caspian Middle Corridor (BTK) with Caspian ferry visualised separately
- RoRo: Çeşme–Trieste short-sea
- Road: Balkan corridor and European motorways
- ISO 14083 / EcoTransIT Well-to-Wheel CO₂ model
- Interactive Leaflet map with correct water/land routing

## Open
Open `index.html` in a browser (internet connection required for CDN dependencies).
